# 2026-04-08 下一步计划

## 优先级 1

### 1. 把路书里的推荐做成“可搜、可选、可加入”

目标：
- 不直接推荐结果。
- 先搜一批候选。
- 用户自己选。
- 选中后加入当天路线或保留到清单。

建议做法：
- 保持推荐先在工作台侧栏中完成，不先进地图主链路。
- 优先打磨高德搜索的可用性：
  - 搜索词
  - 城市/区域约束
  - 候选列表展示
  - 结果去重
  - 加入当天路线

验收标准：
- 在任意一天的路书中，用户能搜到一批真实候选点。
- 候选点能被加入当天路线。
- 不影响地图主显示稳定性。

## 优先级 2

### 2. 打磨“推荐 -> 路线编辑”的闭环

目标：
- 从推荐面板选中的点，能明确进入路线编辑逻辑。
- 后续可以决定插入到路线中的哪个位置，而不只是追加到末尾。

建议做法：
- 先支持：
  - 加到末尾
  - 收藏到当天候选清单
- 再支持：
  - 插入到第 N 个点之后
  - 加入后重新规划路线
